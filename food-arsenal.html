Web Project: Recipe Arsenal

Live Project

Technologies: JavaScript, Node, WebPack, Babel

Food Arsenal is a Web Application for searching different types of recipes based on the user input. The user can enter a food query such as soup, pizza, or cheese, and the application will fetch data from an API with curated results. The user than can see the ingredients that make up the recipe and customize how many servings they want, and lastly, the user can add the item to their cart to simulate a purchase.
